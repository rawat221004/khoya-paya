# CCTV demo footage

Drop a video clip named **`demo.mp4`** in this folder to have it play inside the
**🎥 CCTV Tracking** panel on any case detail page (`/cases/[id]`).

```
public/cctv/demo.mp4
```

The tracker works without a clip too — if `demo.mp4` is absent, the panel falls
back to showing the case photo with an animated CCTV "scan" overlay. Either way,
the camera sighting trail (camera id, distance, time, confidence) is generated by
the backend (`/api/cctv/track`) from the real Nashik CCTV camera network, and the
subject's appearance is described from the case photo by Claude vision when an
`ANTHROPIC_API_KEY` is set.

Clicking a sighting in the trail seeks the video to that point.
